# Raine Engine
A hobby rendering - and hopefully video game - engine.
Created by *Gabriel Nakamoto* with **minimal** dependencies using:

- OpenGl
- tinyobjloader
- stb_image
- glfw
- glad
- imgui

## Features
- Model loading (with textures and indices)
- Shader hot loading
